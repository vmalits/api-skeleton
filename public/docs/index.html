Introduction

Base URL: http://localhost

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Auth

POST api/v1/auth/register

Example request:
curl --request POST \
    "http://localhost/api/v1/auth/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --data "{
    \"first_name\": \"John\",
    \"last_name\": \"Doe\",
    \"email\": \"john@doe\",
    \"password\": \"yqy5bjd7vkg@ckm5THG\",
    \"password_confirmation\": \"yqy5bjd7vkg@ckm5THG\"
}"

const url = new URL(
    "http://localhost/api/v1/auth/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

let body = {
    "first_name": "John",
    "last_name": "Doe",
    "email": "john@doe",
    "password": "yqy5bjd7vkg@ckm5THG",
    "password_confirmation": "yqy5bjd7vkg@ckm5THG"
};

fetch(url, {
    method: "POST",
    headers,
    body: JSON.stringify(body),
}).then(response => response.json());

Request      

POST api/v1/auth/register

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Body Parameters

first_name   string   

First Name. Must be at least 2 characters. Must not be greater than 255 characters. Example: John

last_name   string   

Last Name. Must be at least 2 characters. Must not be greater than 255 characters. Example: Doe

email   string   

Email. Must be a valid email address. Example: john@doe

password   string   

Password. Example: yqy5bjd7vkg@ckm5THG

password_confirmation   string   

Confirmation Password. Example: yqy5bjd7vkg@ckm5THG
